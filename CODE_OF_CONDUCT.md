# Code de conduite

## Notre engagement

Dans l'intérêt de favoriser un environnement ouvert et accueillant, nous nous engageons à faire de la participation à ce projet une expérience sans harcèlement pour tout le monde.

## Nos standards

Comportements qui contribuent à créer un environnement positif :
- Utiliser un langage accueillant et inclusif
- Respecter les différents points de vue et expériences
- Accepter gracieusement les critiques constructives
- Se concentrer sur ce qui est le mieux pour la communauté

## Application

Les comportements abusifs peuvent être signalés en contactant le mainteneur du projet via LinkedIn : [linkedin.com/in/ange-awala-57aa1b363](https://www.linkedin.com/in/ange-awala-57aa1b363/)

Ce Code de Conduite est adapté du [Contributor Covenant](https://www.contributor-covenant.org), version 2.0.
