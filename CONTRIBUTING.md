# Contribuer à MathxCanvas

Merci de l'intérêt pour ce projet ! Voici comment contribuer.

## Types de contributions bienvenues

- 🐛 **Corrections de bugs** dans les animations existantes
- 📱 **Améliorations du support tactile/mobile**
- 🎨 **Variantes visuelles** d'une animation existante
- 📖 **Améliorations de documentation** dans les README par jour
- ✨ **Nouvelles animations** dans l'esprit du projet (maths + UX + canvas)

## Processus

1. Fork le dépôt
2. Crée une branche : `git checkout -b feat/nom-de-ta-contribution`
3. Commite tes changements : `git commit -m "feat: description claire"`
4. Push : `git push origin feat/nom-de-ta-contribution`
5. Ouvre une Pull Request vers `main`

## Convention de commits

```
feat:     nouvelle animation ou fonctionnalité
fix:      correction de bug
docs:     modification de documentation
style:    formatage, sans changement de logique
refactor: restructuration de code
```

## Philosophie du projet

Chaque animation doit :
- Utiliser **vanilla JS uniquement** (pas de framework)
- Fonctionner sur **souris ET tactile**
- Avoir un **README** avec le concept math + le principe UX + les valeurs à modifier
